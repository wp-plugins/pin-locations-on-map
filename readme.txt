Now its easy to show your locations with all the related info on website by use of Pin Locations on Map Plugin.
== Description ==

By use of Pin Locations on Map plugin, users can add multiple addresses by cliking on the appropriate map location and fill the form  for that location add image,addresses.

After saving all the locations. by use of shortcode : [PLOM_DISPLAYMAP]. you can show that organised Map at any location of your website.

The Map also suitable for responsive sites as well.

Major features in Pin Locations on Map include:

* you can add as many as possible locations on Map.
* It is very user friendly.

== Installation ==

Upload the Pin Locations on Map plugin to your blog, Activate it and Done.

Start adding Locations by clicking on your locations on Map.
